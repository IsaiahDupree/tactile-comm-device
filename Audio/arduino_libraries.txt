Arduino Libraries Required for Tactile Communication Device
==========================================================

Install these libraries through the Arduino IDE Library Manager:

1. **Adafruit_PCF8575** (by Adafruit)
   - Version: Latest
   - Provides I2C port expander support for 32 buttons
   - Dependency: Adafruit_BusIO

2. **Adafruit VS1053 Library** (by Adafruit) ðŸŽµ NEW!
   - Version: Latest  
   - Controls the VS1053 codec for superior audio quality

   - Hardware interrupt support for background playback

3. **DFRobotDFPlayerMini** (by DFRobot) - OPTIONAL
   - Version: Latest  
   - Only needed if using DFPlayer Mini version
   - Use VS1053 version for better quality

3. **SD** (by Arduino, SparkFun)
   - Version: Latest (usually pre-installed)
   - Manages SD card file operations
   - Required for configuration storage

4. **SPI** (by Arduino)
   - Version: Latest (usually pre-installed)
   - Required for SD card communication

5. **Wire** (by Arduino)
   - Version: Latest (usually pre-installed)
   - I2C communication for PCF8575 expanders

6. **SoftwareSerial** (by Arduino)
   - Version: Latest (usually pre-installed)
   - Serial communication with DFPlayer Mini
