## Priority System
- **HUMAN_FIRST Mode**: Personal recordings (REC) play first, then TTS
- **GENERATED_FIRST Mode**: TTS plays first, then personal recordings
- **Mode Switch**: Triple-press Period button within 1.2 seconds
